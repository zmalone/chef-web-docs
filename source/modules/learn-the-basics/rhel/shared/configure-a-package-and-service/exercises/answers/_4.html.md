```bash
# ~/chef-repo
$ curl -I localhost

```

Here's the code that stops and disables the Apache service.

```ruby
# ~/chef-repo/webserver.rb
service 'httpd' do
  action [:stop, :disable]
end
```

Your modified <code class="file-path">webserver.rb</code> file looks like this.

```ruby
# ~/chef-repo/webserver.rb
package 'httpd'

service 'httpd' do
  action [:stop, :disable]
end

file '/var/www/html/index.html' do
  content '<html>
  <body>
    <h1>hello world</h1>
  </body>
</html>'
end
```
